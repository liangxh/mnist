# Project 1

# 引言

人工神經網絡近數十年來的急速發展, 讓人們意識到人工智能的潛力. 如卷積神經網絡(CNN)在圖像識別領域就有顯著的成果, 並漸漸投入包括人臉識別等實際應用中, 又如反饋神經網絡(RNN)在機器翻譯和語音模仿等方面的突破, 鼓舞著這些領域的研究者. 雖然目前人工神經網絡在不同任務中都達到了優秀的性能表現, 但其中存在著一個核心的問題, 那就是計算機的能源消耗. 複雜的網絡結構能夠大大提升模型的性能, 但這意味就龐大的計算量, 也意味著相當的能源消耗. 而近年來推出基於spike的仿神經硬件則展現出其在節能上的優勢, 並在性能上逼近現有的神經網絡. 為此, 本文針對SNN的性能進行研究. 本文的實驗基於MNIST手寫數字進行識別.

# 方法

本文除了實現Spiking Neural Network (SNN)來觀察它的性能, 另外也實現了Multi-Layer Perceptron (MLP)和CNN進行相互比較.

## 模型

1) MLP

本文實現一個兩層的MLP, 兩層的神經元數量依次為64和128, 激活函數分別嘗試了 Logistic Sigmoid 和 Rectified Linear Units (ReLU). 輸出層是帶10個神經元的Softmax層.

2) CNN

本文實現一個兩層的CNN.　第一層的卷積核大小為3x3, 特征圖數量為4, 激活函數為tanh, 再經過比例為0.25的Dropout層以及Max Pooling. 第二層的卷積核大小同樣為3x3, 特征圖數量同為8, 激活函數同為tanh, 再經過Max Pooling.　最后輸出層同樣是帶10個神經元的Softmax層.

3) SNN

本文實現一個基於integrate-and-fire(IF) neurons的SNN. 参考P. U. Diehl等人[1] 提出的方法實現一個SNN, 首先實現一個基於ReLU的MLP, 再把MLP中原本的神經元替換成IF神經元, 以相同的矩陣參數作為IF神經元的權值.

## 數據與預處理

本文的的實驗基於MNIST手寫數字進行識別, 實驗數據在http;//yann.lecun.com/exdb/mnist/上獲取. 其中訓練數據集包含圖片60,000張, 測試集包含圖片10,000張. 每張圖片為28x28大小, 數值在[0, 255]上的矩陣, 對應一張黑白圖, 其中數據越接近0, 像素點的顏色越接近黑色, 反之則接近白色. 另外每一張圖的標籤是0, 1, ..., 9中的一個數字, 表示圖中顯示的數字.

模型的輸入是一個784維的向量, 對應圖片28x28個像素, 此處把原本在[0,　255]的灰度值線性映射到[0, 1]上, 輸出是一個0, 1, ..., 9之間的一個整數.

## 訓練與測試

神經網絡中的權重均為随機初始化, 在訓練過程中共訓練30輪, 批大小為128. 選擇了交叉熵作為作價函數,　優化方法為RMSprop. 最后取在訓練集上準確率最高的一組權重來對測試集進行測試. 其中評測標準為分類的準確率.
